# gofast-server

### Getting started
 1. get [node.js](https://nodejs.org/en/download/)
 2. get yarn from your OS package manager or install via npm
    - `npm install -g yarn`
 2. install the dependencies
    - `yarn install`
 3. build for production
    - `yarn build`
 4. run dev flow
    - `yarn run`
    - open `localhost:3000` (You'll need a server running to use this)
 5. run tests
    - `yarn test`
